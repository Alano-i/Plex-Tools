# Plex-SortTittle MR插件
## 功能介绍
- [x] 修改Plex排序名称为拼音,以便于进行中文影音的搜索及排序 
- [x] 整理影片标签,将英文标签转换为中文标签
- [x] 自动打标签,自动识别 豆瓣/IMDB TOP 250 影片自动添加标签 方便分类查询

#### 支持类型  
✅ 电影    ✅ 剧集    ✅ 音乐
